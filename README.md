# email-sends-service

## проект предназначен для управления email-рассылками

## Сущности

- клиент    
- рассылка
- сообщение рассылки
- попытка рассылки

## Интерфейс

### Интерфейс системы содержит следующие экраны: 
- список рассылок
- отчет проведенных рассылок отдельно
- создание рассылки
- удаление рассылки
- создание клиента
- удаление клиента
- редактирование клиента
